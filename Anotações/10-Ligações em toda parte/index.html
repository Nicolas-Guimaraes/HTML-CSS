Hyperlinks

Todas vez que clicamos em um local para ir a outra página, outro site ou baixar um arquivo estamos interagindo com um hyperlink.

Para criar um usamos a tag <a>, com o atributo href, que referencia o hipertexto desejado e tem como valor a url desse hipertexto.

Outros dois atributos interessantes são o hreflang e target, respectivamente:

hreflang
Indica a lingua do site linkado.
target
Controla como o hyperlink vai abrir. Por padrão o link abre na mesma janela do site, que é desejado quando uma outra página do nosso site que está linkada, porém não quando o link é outro site, por exemplo. Para controlar isso os valores são:
  • self(padrão)-abre o link na janela atual
  • blank-abre o link em uma nova janela.
  • top- desfaz os frames e abre o destino no navegador completo
  • parent- similar ao top em uma referencia a janela mãe.
  • nome-do-frame caso esteja usanod frames, indicar o nome da janela ao abrir.
    • (Deletado porque o uso de frames é uma tecnica em desuso)

Outro atributo bem interessante é o rel, o qual mostra a relação entre os documentos, e consequentemente o tipo de destino do link. Ele possui varios valores, no contexto da página alguns interessantes são:

  • next-linka para a próxima parte do documento atual.
  • prev-linka para a parte anterior do documento atual.
  • author-linka para o atuor do artigo atual
  • external-linka para um site que não faz parte do site atual.
  • nofollow-linka para um site não endossado, como um link pago.

OBS: Quando linkar o site externo é necessario indicar a url completa, incluindo protocolo e ser for necessario o caminho à uma página específica.
