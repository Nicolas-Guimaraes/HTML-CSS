Simbolos

Para fazer os simbolos <Menor que(less than) e maior que(greater than)>.
Finalmente sei representar uma tag.

Outros simbolos que podem ser exibidos usando códigos:
 
®
©
™
∑
Δ
Se quiser mais é so pesquisar por HTML entities.

Emojis

Emojis também possuem codigos para serem exibidos.
Para pesquisar os codepoints é só usar o site emojipedia.
Ex:
😎 🌹 🔥
Para usar é só substituir o "U+" por "&#x".
