CSS

Cascading Style Sheets is used to format a layout of a webpage.
With it you can control the color, font,the spacing between elements, how a elements are positioned and laid out, what background images or color are to be used, different displays for different devices and screens sizes,etc.

Cascading: "a style applied to a parent element will also apply to all children elements within the parent, unless you specify something else."

Inline CSS

Unique style to a single element, using the style attribute of the element,as we are used to.

Internal CSS

Defines a style for a single HTML page in the "head" section, within a "style" element. Wich is what are we doing for the first time in this document.

External CSS

Defines a style for many html pages, because ins't a part of a HTML document.
To use add a link to it within the "head" section of the html page.

The external sheet file must not contain any hmtl code, and also be saved as a ".css" extension.
