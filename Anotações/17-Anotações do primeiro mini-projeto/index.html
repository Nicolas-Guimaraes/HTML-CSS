Anotações do primeiro mini projeto

Para começar a planejar um projeto a primeira coisa a fazer é o wireframe, planejar qual a estrutura e o comportamento do site. Ele vai servir para planejar as caixas que farão parte da página.

Responsividade para sites é quando a interface se adapta para o tamanho da tela(viewport), desde o contéudo principal, como textos, imagens e menus. Se quiser ver a quantidade de pixels de uma página use as ferramentas do google, mas se quiser uma forma mais pratica use a extensão window resizer.
