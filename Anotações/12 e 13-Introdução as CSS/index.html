12-Trabalhando com estilos

Para evitar problemas com acentuação adicione no inicio do documento a regra em CSS @charset "UTF-8"; a qual indica a compatibilidade de codificação com o padrão UTF-8.

Para adicionar comentarios nas CSS basta usar "/* */".

CSS inline style sempre vai sobrepor CSS internal style.
CSS internal style sempre vai sobrepor CSS external style.

Ainda sim, é bom lembrar:
CSS externo=sempre que puder
CSS interno= pequenas alterações, no máximo.
CSS inline= esquece que existe, usa pra teste estorando.

Um desse tópicos que vou detalhar é como usar gradientes.
A primeira coisa interessante é que se usa "background-image" ao ínves de "color".
Depois define como vai ser o gradiente, nesse caso usei o linear, mas poderia usar um "radial-gradient", e nos valores colocar seu angulo ou direção("to right, to left") e finalmente as cores, o gradiente muda dependendo da ordem.
OBS: se usasse o radial, colocaria "circle" no lugar do ángulo.

Dicas

Algo muito interessante é tomar cuidado com o background preto e letras brancas, por gerar um cansaço visual grande, sites como o da netflix usam isso, porém possuem pouco conteúdo.

Quando estiver decidindo uma paleta de cores, lembre das cores preto e branco, já que provavelmente a página vai ser branca e a letra preta.
Para começar a decidir a paleta primeiro busque uma cor pela emoção que ela passa e a partir disso busque as outras.
Outra forma de decidir é se a logo do cliente tem uma marca cor primaria muito forte, decidindo as outras a partir dela.
Geralmente uma paletra varia de 3 à 5 cores.
