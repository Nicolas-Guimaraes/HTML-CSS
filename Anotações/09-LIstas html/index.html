Listas na html

A HTML tem 3 tipos de listas: <ol>, lista ordenada; <ul>, lista não ordenada e <dl>, lista de definição.

Lista ordenada e não ordenada

Ambas usam a tag <li> para identificar cada item.

Na tag <ol> o parâmetro "type" para configurar o tipo de marcador da lista, com os seguintes valores:

  • 1- Cria listas numeradas
  • A- Cria listas alfabéticas em maiúsculas
  • a- " " em minúsculas
  • I-Cria listas com algarismos romanos em maiúsculas
  • i-"" em minúsculas

E o valor do parâmetro "start" define onde começa a contagem

Já na tag <ul> o parâmetro "type" tem os seguintes valores:

  • disc(padrão)- bola preta totalmente pintada
  • circle- bola preta sem preenchimento
  • square-quadrado preto com preenchimento

Listas de definição

A lista de definição é um pouco diferente, pois usa tags diferentes e não possui marcadores, ela usa a tag <dl> para definir a lista, os termos são definidos pela tag <dt> e a descrição é definada pela tag <dd>:

míngua
pobreza extrema; indigência, penúria.
hemeroteca
setor das bibliotecas onde se encontram coleções de jornais, revistas, periódicos e obras em série.
