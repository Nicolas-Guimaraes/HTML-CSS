Tableless: No passado, com a HTML4, as tabelas foram usadas para fazer a estruturas dos sites. Oque não deve ser feito nunca mais, porém elas também não devem ser demonizadas, apenas serem usadas para sua função original de organizar dados

Primeira exemplo de tabela:

População das unidades Federativas
Cidade População
São Paulo 45.000.000
Minas Gerais 30.000.000
Rio de Janeira 30.000.000
Bahia 30.000.000

Essa tabela não foi construido como indica o material, pois fiz no olho usando as tags apresentadas antes.
No material é indicado usar as tags semânticas <thead>, <tbody> e <tfotter>, as quais geralmente são usadas em tabelas maiores.
Outra coisa que errei é que a tag <th>, marca o título, então ela deveria estar sendo usada em cidade e população.
