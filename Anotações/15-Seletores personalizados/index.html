Seletores personalizados

Podemos identificara elementos HTML únicos(não se pode usar o mesmo id duas vezes) com um id(#) ou agrupar elementos multíplos com caracteristicas semelhantes com um class(.)

OBS: nomeie suas classes ou ids por funcionalidade.

Propriedades herdadas

Da mesma que forma que as CSS externa é sobreposta pela inline os seletores personalizados vão adicionar as suas propriedades e se tiverem as mesmas propriedades elas seram sobreescritas. Como acontence na CSS desse documento, as tags h1, h2 e h3 estão com as propriedade color: "black", mas ela está sobreposta nos dois casos.

OBS: Você pode adicionar mais de uma class em um elemento html ou adicionar um id e uma class no mesmo elemento html.

Pseudo-classes e pseudo-elementos

A pseudo-classe CSS é uma palavra chave usada marcada por ":", que se adiciona as declarações de um seletor e indica um estado em especifico do elemento, como exemplo:
:hover; :visited; :empty; :checked; :active e :focus.

Já a pseudo-elemento CSS é uma palavra chave marcada pelos sinais "::", que se adiciona as declarações de um seletor e referencia uma parte especifica desse elemento, por exemplo: ::first-letter; ::first-line; ::before; ::after.
