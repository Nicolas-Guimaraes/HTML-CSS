Imagens

Devido aos direitos autorais não se pode usar qualquer imagem quando se está produzindo algo para uso
comercial. Portanto para conseguir imagens que podem ser usadas podemos fazer algumas coisas:

- Produzir conteudo proprio.
- Conseguir autorização legal de quem detem os direitos autorais.
- Comprar os direitos das imagens em sites como:
 Istockphoto
 depositphotos
 shutterstock
- Usar imagens de domínio publico.
Para pesquisá-las você pode usar:
 google imagens com o filtro Marcadas para reutilização com modificação.

 wikimedia commons
 pexels
 pixabay
E muitos outros...
Mas lembre de ler bem as instruções para o uso da imagem

Formato de imagens

Percebeu o quão cagada a qualidade da imagem de exemplo do google está?
Então, muito provavelmente isso aconteceu porque a qualidade de compactação dela é bem baixa.
Os dois formatos de imagem mais usados são o JPEG e PNG.
As vantagens do JPEG é que ele é leve, mas tem que tomar cuidado com a compactação pra imagem não ficar borrada, o ideial é entre 30% e 50%.
Já o PNG é mais pesado, mas permite configurar a transparencia dos pixels, permitindo imagens com fundos transparentes.

Tamanho das imagens

Regra: Sempre use imagens do tamanho certo, nada de ficar mudando o seu tamanho com codigos de CSS.
Porque o mecanismo de busca do google penaliza sites lentos e pesados, e diminuir a imagem com codígos não a deixa mais leve.

Favicon

Para usar o Favicon você precisa ter um arquivo no formato .ico você pode:
pegar algum pronto no site iconarchive.
Pode desenhar um no site favicon.cc.
Ou converter uma imagem no site Favicon.io.

Depois disso, salve na pasta o arquivo .ico na pasta do projeto.
Para usar basta usar a tag <link:favicon> na área <head>, como está acima no documento.
Outros formatos além do .icon são aceitos, como PNG e SVG, o qual seria a melhor opção por ser vetorizado, porém nem todos os navegadores o aceitam.
