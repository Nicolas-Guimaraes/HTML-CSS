Assunto principal

Um título deve seguir uma hierarquia:
<h1> é assunto principal, oque não significa que só deve ser usado uma vez, já que uma página pode ter vários asssuntos principais.
<h2> é o sub-assunto de <h1>, é assim por diante....
É algo simples, mas preste atenção para não cometer erros bobos, porque hierarquia é importante.
