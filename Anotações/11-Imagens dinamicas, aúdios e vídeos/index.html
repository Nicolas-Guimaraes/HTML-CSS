Imagens dinamicas, aúdios e vídeos

Imagens dinamicas

Um conteúdo de sites deve se adaptar a tamanhos de tela diferentes, o contéudo que mais sofre com isso são as imagens, podemos redimensiona-lás com as css, mas isso não mudara o seu tamanho, consequentemente aumentando o consumo de dados e deixando o site lento.

Para criar imagens flexiveis usaremos as tags <picture> e <source>:
A tag <picture> serve para representar suas crianças, dentro dela colocaremos as tags <source> e <img>. A tag <source> deve respeitar uma ordem crescente dos valores nos atributos "max-widht" e a tag <img> deve sempre vir por último
E a tag <source> indica a mídia alternativa para outro elemento de midia. Ela possui 3 atributos:

  • type-indica o tipo de media.
  • srcset-indica o arquivo a ser carregado quando o tamanho indicado pelo atributo "media" é atingido.
  • media-Indica o tamanho maximo, com a propriedade "max-width", a ser considerado para carregar a imagem indicada no atributo "srcset".

OBS: Colocar um valor ligeiramente acima do tamnho da imagem na propriedade "max-widht" gera resultado mais interessantes.

Aúdio

Para colocar um aúdio em nosso site basta usar as tags <audio> e <source>.
A tag <audio> tem alguns atributos:

  • preload-indica se o aúdio será pre-carregado e tem três valores:
    • metadata-carrega apenas as informações sobre o arquivo(tamanho, tempo, informações de direito)
    • none-não carrega nada até que o úsuario ou um script inicie a reprodução.
    • auto-carrega o arquivo inteiro assim que a página é carregada.
  • controls-apresenta o player na tela, para que o úsuario possa interagir com ele.
  • loop-vai fazer que o aúdio se repita quando terminar.
  • autoplay-inicia a reprodução do aúdio assim que a página é carregada.

Já a tag <source> só usa atributos "src" e "type", que já estamos acostumados.

OBS: Coloque o <source> com seu formato de aúdio favorito primeiro, caso ele falhe os outros irão carregar.

Vídeo

Basta usar a tag <video>. Seus atributos são:

  • width-indica a largura do vídeo
  • poster-indica a imagem que aparece antes de apertar o play
  • Autoplay e controls- As mesmas funções descritas no áudio.

OBS: Arquivos de vídeo não são tão simples quanto os outros, nos quais o formato indica o padrão para abrir e reproduzir a mídia. Vídeos possuem formatos e codecs(dispositivo de hardware ou software que decodifica sinais).
Os formatos mais aceitos são MPEG, WEBM e OGG, com os 2 primeiros tendo mais compatibilidades.

Porém, hospedar os proprios vídeos é pesado e caro porque quando mais pessoas acessam o site o vídeo consome uma banda desgraçada.
Para contornar isso podemos hospedar nossos vídeos no youtube e apenas incorpora-lós na HTML. Para fazer isso basta clicar no botão "compartilhar" e "incorporar".
